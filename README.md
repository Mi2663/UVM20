# Учебная Виртуальная Машина (УВМ) - Вариант №20

## Описание проекта
Проект по разработке ассемблера и интерпретатора для учебной виртуальной машины в рамках практической работы.

## Архитектура УВМ
- Размер команды: 3 байта
- Формат команд: [A:4 бита | B:12-13 бит]
- Память: раздельная для команд и данных

## Поддерживаемые команды
1. **LOAD_CONST** (A=10) - загрузка константы в аккумулятор
2. **LOAD_MEM** (A=0) - загрузка из памяти
3. **STORE_MEM** (A=14) - запись в память
4. **SQRT** (A=2) - вычисление квадратного корня

## Этапы разработки
1. Ассемблер (парсинг в промежуточное представление)
2. Генерация машинного кода
3. Базовый интерпретатор с памятью
4. Реализация АЛУ (команда sqrt)
5. Тестовая задача (обработка вектора)
6. Кроссплатформенное GUI приложение
